# event-api
Flask Event Management Rest Api

### Installation

```
bash install.sh
```

### run
```
bash build.sh <env>
<env: prod | dev>
```

### Api
Api documentation

#### Create Event Type
```
http://localhost:5000/event/type
METHOD: POST
```

###### Request Body
```
{
    "name": "Meetup Event",
    "description": "Friends get together"
}
```

###### Response Body
```
{
    "status": 200,
    "message": "Event type created successfully",
    "data": {
        "description": "Friends get together",
        "name": "Meetup Event"
    }
}
```


#### Get Event Type
```
http://localhost:5000/event/type
METHOD: GET
```

###### Response Body

{
